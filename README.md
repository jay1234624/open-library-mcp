# Open Library MCP Server

## Overview

This project is a Model Context Protocol (MCP) server built with TypeScript. It connects AI assistants to the Open Library API, allowing them to search books, retrieve book details, fetch author information, and list book editions.

## Features

- Search books

- Get work details

- Get author profile

- Get book editions

## Technologies Used

- TypeScript

- Node.js

- Model Context Protocol (MCP)

- Open Library API

## Installation

```bash

npm install

```

## Build

```bash

npm run build

```

## Run

```bash

npm start

```

## Available MCP Tools

- `search_books(query)`

- `get_work_details(workId)`

- `get_author_profile(authorId)`

- `get_editions(workId)`
